Getting Started
===============

New to ``brainmass``? Start here. Install the package, run your first simulation through
the ``Simulator``, build the mental model behind Step-models / ``Simulator`` / ``Network``
/ ``Fitter`` and unit-safe quantities, then pick the persona learning path that matches
your goals.
